Create your Custom maintenance mode

== Description ==

In Custom maintenance mode, you need to enable maintenance mode. 

= There are two options: =
1. Take user to the specific WP page during maintenance mode, select from drop down.
2. Give URL for count-down page or define Custom URL. 

Subscribe functionality include in custom mainatenance mode plugin and access admin user of front end. 

If you select option 2, then you can fill the information in fields like 
 Meta Title, Meta Description, Logo Images/Text, Display Logo Images,  Custom Title for under construction, Enter Subtitles, Contact Email, End date for expiring the under construction/maintenance mode. 

= Features: =
1. Easy to use and customize mode
2. Countdown timer
3. Upload own logo or text
4. Redirect Page
5. Social media profiles
6. Collect subscriber email
7. Subscriber with validation
8. SEO friendly

== Installation ==

This section describes how to install the plugin and make it working.

1. Upload `custom-maintenance-mode` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Alternatively, you can install and activate the plugin through the WordPress plugin manager.
 
For Information: When you install first time, folder will be created in Root directory.
